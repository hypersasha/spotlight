# VK Mini Apps Starter Pack

## How to run
Firstly, you have to download all dependencies.

`npm install`

To run VK Mini App for development

`npm start`

To build VK Mini App for deploy

`npm run predeploy`

To publish App to GitHub Pages

`npm run deploy`
